Question 1

ggplot(mpg, x = displ, y = hwy) + geom_point()
## Error: geom_point requires the following missing aesthetics: x and y

a) briefly describe what causes the error

In order to map the variables to the plot, the users should define the variables in the aes() argument. However, this user didnt do that. But he tried to plot the mapping variables by using geom_point(). So ggplot was asked to plot the dots but it wasnt told what data should be plotted. And this created the error.

b) modify the code to fix any problems.

ggplot(mpg, aes(x = displ, y = hwy)) + geom_point()
